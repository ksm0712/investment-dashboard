# Investments Dashboard

Streamlit dashboard for tracking investments, refreshing stock and mutual fund prices, and viewing portfolio allocation by country and asset type.

## Run locally

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run dashboard.py
```

Local runtime files such as `.env`, `venv/`, logs, and `investments.db` are intentionally ignored by git.
